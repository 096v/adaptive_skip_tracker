## 六、分分辨率性能

| 分辨率 | 关键帧耗时 | 跳过帧耗时 | 加速比 |
|--------|:---:|:---:|:---:|
| 640x360 | ~20 ms | ~8 ms | 2.5x |
| 640x480 | ~10 ms | ~3 ms | 3.3x |
| 960x540 | ~12 ms | ~4 ms | 3.0x |
| 1920x1080 | ~85 ms | ~27 ms | 3.1x |

跳过帧耗时包含: 灰度转换 + LK 光流 + 仿射变换 + 卡尔曼预测。加速比随分辨率升高而增大（YOLO 推理与像素数 O(n^2) 成正比，光流为 O(n)）。

---

## 七、追踪稳定性

全部 50 个视频在所有间隔下：

- 跳过帧期间追踪数**恒定不变**（任意 4 帧跳过序列中 0 衰减）
- 关键帧恢复：场景切换后 2-3 个关键帧内追踪数回弹
- 16 个合成视频：960 帧多样化运动模式（直线、正弦、圆周、之字、缩放、交叉、遮挡、静止）零崩溃
- 整体保持率：**99.4%**（间隔=5），**98.1%**（间隔=10）

## 八、关键 Bug 修复

| 修复 | 效果 |

| 空间匹配 (Fix #1) | MOTA +13pp，消除 Track 与偏移的错配 |

| 无效特征点清理 (Fix #3) | 减少噪声，提升仿射估计精度 |
| 放宽匹配方向修正 | IDF1 +15pp，IDSw 可控 |
| 累计尺度钳位 (Fix #6) | 防止跳过序列中尺度漂移 |

---

## 九、已知限制

1. **极小目标 (< 20px)**: LK 特征点不足，退化为纯卡尔曼预测
2. **剧烈光照变化**: 光流亮度恒定假设失效
3. **连续跳帧 >10**: MOTA 开始明显下降，建议间隔 5-7
